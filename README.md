## Installation 🍀

- Source 🍙

```bash
$ git clone --depth=1 https://github.com/pwnwriter/rayso
$ cd rayso
$ cargo build --release # move rayso binary to your any $PATH 🥦
```

- Cargo 🦀

```bash
$ cargo install rayso
 ```

- [METIS Linux](https://metislinux.org)

```bash
$ sudo/doas pacman -S rayso
```

- Aur repository

```bash
$ paru/yay -S rayso
```
